# simple-movie-website
 
# 概要

用於學習RWD響應式網頁設計

基於HTML、Tailwind CSS、JavaScript 的電影網站可以瀏覽電影信息和進行訂票

# 環境配置

開發環境：Windows 10　，　Visual Studio Code

# 專案結構
## 前端 
HTML、JavaScript、Tailwind CSS

---------------------------------------
# 主頁面

利用Bootstrapp的輪播功能來顯示電影海報
